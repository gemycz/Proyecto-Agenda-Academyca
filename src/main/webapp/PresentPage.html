NOSOTROS

¿QUÉ ES STUDAY?

StuDay es un sistema web basado en una Agenda Académica para el estudiante universitario común. Su back-end esta desarrollado con el framework JSF el cual aplica el Modelo Vista Controlador (MVC) para su funcionamiento, ademas utiliza el SGBD MariaDB para el almacenamiento de datos.Como parte del Front-End aplica el marco JSF potente y liviano llamado BOOTSFACES el cual esta basado en Bootstrap 3 y jQuery.

Docentes, Asignaturas & Tareas

Este sistema cuenta con Tres módulos base para el funcionamiento su funcionamiento lógico. En el módulo de docentes se registrarán datos como el nombre, apellido, correo y teléfono del docente, además de mostrar una lista de los docentes registrados. El módulo de Asignatura permite registrar las asignaturas del estudiante y mostrar una lista de las que estén registradas. Por ultimo el modulo asignatura integra el registro de las actividades del estudiante basándose en las asignaturas y docentes que las pueden involucrar.

Agenda

La Agenda integra paneles en los cuales se muestran las tareas pendientes segun la fecha registrada en los mismos y el día lo cual permitira un mejor control de lo el estudiante tiene que hacer en la semana.
